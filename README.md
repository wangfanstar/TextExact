# extract.html — 文本行提取工具

一个纯前端、无需安装、直接用浏览器打开的文本行提取工具。
所有处理均在本地完成，文件内容不会上传至任何服务器。

---

## 快速开始

直接用浏览器打开 `extract.html` 即可，无需服务器或任何依赖。

---

## 界面布局

```
┌─────────────────┬──────────────────────────────────────┐
│   使用说明      │  文件来源                             │
│  （左侧固定）   │    ├─ 拖拽/点击上传                   │
```

- **右侧**：主操作区（加载文件 → 搜索 → 结果 → 下载）

## 功能说明

### 1. 文件来源（三种方式可混合使用）

#### 拖拽 / 点击上传
- 点击上传区域或将文件**拖拽**到上传框
- 支持 `.c` `.h` `.txt` `.log` `.csv` 等任意文本文件，可批量选择

#### 🌐 网络地址（URL）
- 输入 `http://` 或 `https://` 地址，按 **Enter** 或点击「加载」
- 通过 `fetch()` 拉取远程文本内容
- 需要目标服务器允许跨域访问（CORS）

#### 💾 本地路径
- 输入绝对路径，如 `C:\logs\foo.txt`，自动转换为 `file:///` 协议加载
- 部分浏览器（如 Chrome）对本地文件访问有安全限制，若加载失败请改用拖拽上传
- UNC 网络路径（`\\server\share\...`）建议映射为驱动器后用本地路径，或通过 HTTP 访问后用 URL 方式加载

所有来源加载的文件统一进入文件列表，可单独移除。

---

### 2. 包含关键词

- 在输入框输入关键词后按 **Enter** 或点击「+ 添加」
- 支持添加**多个**关键词，每个以不同颜色标签区分，点击 ✕ 单独移除

**多关键词匹配模式：**

| 模式 | 说明 |
|------|------|
| **OR（任意命中）** | 一行只需包含任意一个关键词即被提取 |
| **AND（全部命中）** | 一行必须同时包含所有关键词才被提取 |

**结果分组：**

| 模式 | 说明 |
|------|------|
| **合并** | 所有匹配行按文件顺序混合输出 |
| **按关键词分组** | 每个关键词单独列出其匹配结果，带分组标题 |

---

### 3. 排除关键词

在包含关键词下方的红色区域中设置排除词，命中排除词的行将从结果中自动剔除。

- 输入排除词后按 **Enter** 或点击「+ 排除」，以红色 ⊘ 标签显示
- 排除词有**独立**的大小写、正则选项（与包含关键词互不干扰）
- 标签角标：`[Aa]` 表示区分大小写，`[.*]` 表示正则模式

**多排除词组合模式：**

| 模式 | 说明 |
|------|------|
| **OR（含任一即排除）** | 命中任意一个排除词即剔除（默认） |
| **AND（全部命中才排除）** | 须同时包含所有排除词才剔除，适合精确排除 |

---

### 4. 搜索选项（包含关键词）

| 选项 | 说明 |
|------|------|
| **区分大小写** | 勾选后 `ABC` 与 `abc` 视为不同字符串 |
| **正则表达式** | 勾选后关键词按正则语法解析 |
| **显示文件名** | 每行结果前加上来源文件名，如 `[foo.c]` |
| **显示行号** | 每行结果前加上原始行号，如 `L42:` |

---

### 5. 结果操作

- 结果显示在可编辑的文本框中，可**直接修改**内容，后续操作均以修改后内容为准
- 点击「**↗ 新标签页打开**」将当前结果在新标签页中展示，便于全屏阅读或复制
- 填写保存文件名后点击「**下载结果文件**」，保存为本地文本文件

## 常用正则速查

| 表达式 | 含义 |
|--------|------|
| `.` | 任意单个字符 |
| `.*` | 任意字符（0 个或多个） |
| `\d+` | 一个或多个数字 |
| `\bword\b` | 精确匹配单词 word |
| `foo\|bar` | 匹配 foo 或 bar |
| `^text` | 行首以 text 开头 |
| `text$` | 行尾以 text 结尾 |
| `[A-Z]` | 大写字母 A–Z 之一 |
| `\s+` | 一个或多个空白字符 |
| `(a\|b)c` | ac 或 bc |
