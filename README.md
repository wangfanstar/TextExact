# extract.html — 文本行提取工具

一个纯前端、无需安装、直接用浏览器打开的文本行提取工具。
所有处理均在本地完成，文件内容不会上传至任何服务器。

---

## 快速开始

直接用浏览器打开 `extract.html` 即可，无需服务器或任何依赖。

---

## 界面布局

```
┌─────────────────┬──────────────────────────────────────┐
│   使用说明      │  上传文件                             │
│  （左侧固定）   │  搜索设置                             │
│                 │  提取结果                             │
└─────────────────┴──────────────────────────────────────┘
```

- **左侧**：使用说明面板，页面滚动时保持固定可见
- **右侧**：主操作区（上传 → 搜索 → 结果 → 下载）

## 功能说明

### 1. 上传文件

- 点击上传区域或将文件**拖拽**到上传框
- 支持 `.c` `.h` `.txt` `.log` `.csv` 等任意文本文件
- 可一次上传**多个**文件，已加载文件以标签形式显示，可单独移除

### 2. 添加关键词

- 在输入框输入关键词后按 **Enter** 或点击「+ 添加」
- 支持添加**多个**关键词，每个关键词以不同颜色标签区分
- 点击标签上的 ✕ 可单独移除某个关键词

### 3. 多关键词匹配模式

| 模式 | 说明 |
|------|------|
| **OR（任意命中）** | 一行只需包含任意一个关键词即被提取，适合收集多个相关主题 |
| **AND（全部命中）** | 一行必须同时包含所有关键词才被提取，适合精确过滤 |

### 4. 结果分组

| 模式 | 说明 |
|------|------|
| **合并** | 所有匹配行按文件顺序混合输出 |
| **按关键词分组** | 每个关键词单独列出其匹配结果，带分组标题，方便对比 |

### 5. 搜索选项

| 选项 | 说明 |
|------|------|
| **区分大小写** | 勾选后 `ABC` 与 `abc` 视为不同字符串 |
| **正则表达式** | 勾选后关键词按正则语法解析，支持完整 JavaScript 正则 |
| **显示文件名** | 每行结果前加上来源文件名，如 `[foo.c]` |
| **显示行号** | 每行结果前加上原始行号，如 `L42:` |

### 6. 编辑与下载

- 结果显示在可编辑的文本框中，可**直接修改**内容
- 填写保存文件名后点击「下载结果文件」，保存为本地文本文件

## 快捷键

| 按键 | 操作 |
|------|------|
| `Enter` | 将当前输入框内容添加为关键词 |
| `Ctrl` + `Enter` | 执行搜索（提取匹配行） |

## 常用正则速查

| 表达式 | 含义 |
|--------|------|
| `.` | 任意单个字符 |
| `.*` | 任意字符（0 个或多个） |
| `\d+` | 一个或多个数字 |
| `\bword\b` | 精确匹配单词 word |
| `foo\|bar` | 匹配 foo 或 bar |
| `^text` | 行首以 text 开头 |
| `text$` | 行尾以 text 结尾 |
| `[A-Z]` | 大写字母 A–Z 之一 |
| `\s+` | 一个或多个空白字符 |
| `(a\|b)c` | ac 或 bc |
